Recuperado desde [https://github.com/rafachavero/CHI-ANALYTICS-PRO](https://github.com/rafachavero/CHI-ANALYTICS-PRO).

## Descripción
Este es un prototipo de la aplicación CHI Analytics. Es una aplicación React de una sola página que se ejecuta directamente en el navegador utilizando Babel Standalone.

## Cómo ejecutar localmente

Debido a las políticas de seguridad de los navegadores (CORS), este proyecto no se ejecutará correctamente si abres el archivo `index.html` directamente (file://). Necesitas usar un servidor web local.

### Opción 1: Usando npx (Node.js)
Si tienes Node.js instalado:

```bash
npx http-server .
```

### Opción 2: Usando Python
Si tienes Python instalado:

```bash
python -m http.server 8080
```

Luego abre tu navegador en `http://localhost:8080`.
